# Dato: 2026-02-16

## Om Prosjektet (PhD i Industridesign/Formgjeving ved AHO)
Dette prosjektet er fundamentet for ein serie på tre PhD-artiklar. Hovudmålet er å radikalt heve kvaliteten på estetiske fag og industridesign, bygd på filosofien "form follows fitness" (Inspirert av Le Corbusier og Jony Ive). Med ein bakgrunn som dataingeniør kombinerast over 100 eigenproduserte 3D-modellar og høgoppløysande bilete frå Nasjonalmuseet for å gjere nyskapande, kvantitative og kvalitative materialanalysar av norske stolar.

## Innhald
- stolar.db          — SQLite-database med alle data
- norske_stolar.xlsx — Excel-fil med alle felt
- norske_stolar.csv  — CSV (semikolon-separert, UTF-8 BOM)
- norske_stolar.json — Rå JSON-data
- bilete/            — Biletfiler (JPG) for kvar stol
